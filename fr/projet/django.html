Fin Octobre - début Novembre 2024 - Projet universitaire - Django (python), Docker, Git

Le projet de Django est un projet universitaire durant le premier semestre de L3, qui consiste à crée un site web pour gérer des collections. J'ai réalisé ce projet avec Nathan Rissot, Agathe Papineau et Joachim Larrouy.

Ce projet est assez rapide car il ressemble beaucoup au TP que nous avons fait, pour ce projet nous avons dû utiliser docker qui était une nouveauté pour moi. Dans ce projet, il fallait crée une petite base de données à l'aide de Django et faire un système de gestion d'une collection, donc ajouter, modifier, supprimer et voir sa collection.
