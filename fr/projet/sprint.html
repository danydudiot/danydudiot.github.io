Décembre 2024 - Projet universitaire (20/20) - HTML, PHP, SQL, JS, Git

Le sprint est un projet universitaire durant le premier semestre de L2, qui consiste à crée un site web pour une agence bancaire pour la gestion des comptes, contrat, RDV. J'ai réalisé ce projet avec Nathan Rissot et Agathe Papineau.

Pour ce projet, nous avons commencé par concevoir la base de données en réalisant le MCD et le MLDR à l'aide de win-design un logiciel installé dans l'université qui nous à permit de générer facilement les commandes SQL pour crée les différentes tables.

Ensuite, nous devions réaliser le projet en respectant le modèle MVC (Modèle/Vue/Contrôleur), je me suis occupé de la partie contrôleur, cette partie m'a permis de renforcer mes connaissances en PHP, je réalisais également un début de page web simple qui était retravailler par Nathan qui s'occupais du design du site, j'ai également aidé Agathe dans la partie du modèle avec les interactions avec la base de données.
